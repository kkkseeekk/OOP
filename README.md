## Консольна програма для виводу аргументів командного рядка

### Постановка задачі
1. Підготувати сховище до розміщення проєкту.
2. Написати просту консольну програму (вивід на екран аргументів командної строки).
3. Прикріпити посилання на GIT.

### Опис проєкту
Ця програма використовує базовий функціонал мови Java для отримання масиву рядків `args[]` та виведення їх кількості та змісту у консоль.
